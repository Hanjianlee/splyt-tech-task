# Splyt Tech Task App

- ### Step 1

  - #### Clone Repository

    git clone https://github.com/Hanjianlee/splyt-tech-task.git

- ### Step 2

  - #### Install Initial Package

    npm install

- ### Step 3

  - #### For Initial run (this step will install all necessary packages)

    npm run all-init

  - #### For subsequent runs

    npm run all
